# LinPomodoro - 系统托盘番茄钟

一个基于 Python 的系统托盘番茄钟应用程序，轻量无窗口，专注提升效率。

## 功能介绍

- 圆角方形托盘图标，根据状态动态变色
- 图标上直接显示倒计时分钟数和状态文字
- 工作时间、短休息、长休息自动循环切换
- 工作时间最后 5 分钟变橙色、最后 1 分钟变红色
- 时间结束时桌面通知提醒
- 自动开始下一个时段
- 右键菜单控制（开始/暂停、下一轮、重置、查看配置、退出）

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行

```bash
python pomodoro.py
```

或双击 `start.bat` 启动（Windows）。

## 操作说明

| 操作 | 说明 |
|------|------|
| 单击/双击图标 | 开始 / 暂停 |
| 右键 → 下一轮 | 跳到下一轮（工作中直接结束，休息中切换到工作） |
| 右键 → 重置 | 重置当前轮的时间 |
| 右键 → 查看配置 | 弹窗显示当前配置信息 |
| 右键 → 退出 | 退出程序 |

## 默认时间

| 时段 | 时长 |
|------|------|
| 工作时间 | 25 分钟 |
| 短休息 | 5 分钟 |
| 长休息 | 15 分钟（每 4 轮触发） |

## 颜色状态

| 状态 | 颜色 |
|------|------|
| 工作中 | 红色 |
| 工作中（≤5 分钟） | 橙色 |
| 工作中（≤1 分钟） | 红色（紧迫） |
| 休息中 / 长休息 | 绿色 |
| 已暂停 | 灰色 |

## 自定义配置

编辑 `pomodoro.py` 顶部的配置常量，修改工作/休息时间、图标大小、颜色方案等参数。

## 技术实现

- **托盘图标**: pystray
- **图标生成**: Pillow (PIL)
- **桌面通知**: plyer
- **计时**: threading + time.sleep
- **弹窗**: ctypes (Windows MessageBox)
